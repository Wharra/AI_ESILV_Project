Gestion des valeurs manquantes¶

On peut voir qu'il y a énormément de valeurs manquantes dans le jeu de données. Si il y'en avait une quantité raisonnable on pourrait les remplacer par la médiane mais dans le cas présent la quantité est énorme. Certaines variables ont plus de 80% de NA ce qui est énorme. Si on comble toutes ces valeurs avec les médiane ou les moyennes cela dénaturera grandement les données. On va donc supprimer les lignes avec plus de 85% de valeurs manquantes.

In [ ]:
data.replace("?", np.nan, inplace=True)

# Calcul du seuil : On garde uniquement les lignes ayant au moins 85% de valeurs non nulles
threshold_rows = int(data.shape[1] * 0.85)
df_cleaned_rows = data.dropna(axis=0, thresh=threshold_rows).copy()

# Réimputation des colonnes restantes : moyenne pour numériques, mode pour catégorielles
for col in df_cleaned_rows.columns:
    if df_cleaned_rows[col].isna().sum() > 0:
        if df_cleaned_rows[col].dtype in [np.float64, np.int64]:
            df_cleaned_rows[col] = df_cleaned_rows[col].astype(float)
            df_cleaned_rows[col].fillna(df_cleaned_rows[col].mean(), inplace=True)
        else:
            df_cleaned_rows[col].fillna(df_cleaned_rows[col].mode().iloc[0], inplace=True)

Interprétation¶

La première composante est fortement influencée par des indicateurs liés à l’usage et l’accès à Internet (comme l'usage individeul) tandis que la deuxième est dominée par des variables liées à la capacité technique et aux infrastructures récentes (comme la 5G). Cela suggère que la première composante capte le niveau global de connectivité numérique d’une région. La deuxième semble refléter le niveau technologique avancé des infrastructures de réseau.

Conclusion¶

L’ACP a permis de résumer l’information contenue dans l’ensemble des indicateurs télécoms en seulement 2 composantes principales, tout en conservant plus de 95 % de la variance totale.

Analyse des résultats :¶

  • La 1ʳᵉ composante est dominée par des indicateurs liés à l’usage numérique (Internet, abonnements, LTE…).
  • La 2ᵉ composante capture des éléments liés à la maturité technologique des réseaux (5G, bande passante, trafic).
  • Les projections et reconstructions dans l’espace d’origine montrent que les données sont fortement compressibles avec très peu de perte d’information.

Utilité de l’ACP dans ce cas :¶

  • Réduction de dimension avant modélisation
  • Visualisation claire de la structure des données
  • Identification des variables les plus explicatives

L’ACP est donc un outil efficace ici pour simplifier et interpréter la complexité des indicateurs TIC tout en conservant l'essentiel de l'information.

import pandas as pd
from sklearn.model_selection import train_test_split, GridSearchCV
from sklearn.svm import SVC
from sklearn.preprocessing import StandardScaler
from sklearn.metrics import classification_report, accuracy_score, confusion_matrix, ConfusionMatrixDisplay
import matplotlib.pyplot as plt
# Création du pivot
df_pivot = df.pivot_table(index=["Region", "Year"], columns="Indicator", values="Value_per100")
df_pivot = df_pivot.dropna(thresh=int(df_pivot.shape[1] * 0.85)).copy()
df_pivot.fillna(df_pivot.mean(), inplace=True)
df_pivot = df_pivot.reset_index()

# Séparation des features et de la cible
X = df_pivot.drop(columns=["Region", "Year"])
y = df_pivot["Region"]

# Normalisation
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# Séparation en ensemble train/test
X_train, X_test, y_train, y_test = train_test_split(X_scaled, y, test_size=0.2, random_state=42)

# Implémentation SVC sans tuning
svc_default = SVC()
svc_default.fit(X_train, y_train)
y_pred_default = svc_default.predict(X_test)

# GridSearchCV pour tuning
param_grid = {
    'C': [0.01, 0.1, 1, 10],
    'degree': [2, 3],
    'gamma': [0.01, 0.1],
    'kernel': ['rbf', 'poly']
}
svc = SVC()
grid_search = GridSearchCV(svc, param_grid, cv=5)
grid_search.fit(X_train, y_train)

# Meilleur modèle après tuning
svc_tuned = grid_search.best_estimator_
y_pred_tuned = svc_tuned.predict(X_test)

# Évaluation
print("Évaluation du modèle SVC sans tuning :")
print(classification_report(y_test, y_pred_default))
print(f"Accuracy : {accuracy_score(y_test, y_pred_default):.2f}")

print("\nÉvaluation du modèle SVC avec tuning :")
print(classification_report(y_test, y_pred_tuned))
print(f"Accuracy : {accuracy_score(y_test, y_pred_tuned):.2f}")

# Matrices de confusion
cm_default = confusion_matrix(y_test, y_pred_default, labels=svc_default.classes_)
cm_tuned = confusion_matrix(y_test, y_pred_tuned, labels=svc_tuned.classes_)

fig, axs = plt.subplots(1, 2, figsize=(14, 6))
ConfusionMatrixDisplay(confusion_matrix=cm_default, display_labels=svc_default.classes_).plot(ax=axs[0], cmap='Blues')
axs[0].set_title("SVC sans tuning")
ConfusionMatrixDisplay(confusion_matrix=cm_tuned, display_labels=svc_tuned.classes_).plot(ax=axs[1], cmap='Greens')
axs[1].set_title("SVC avec tuning")
plt.tight_layout()
plt.show()

Conclusion¶

Le projet met en évidence que certains indicateurs TIC suffisent à caractériser géographiquement une région. L’utilisation combinée de techniques exploratoires, de réduction de dimension et de modèles supervisés a permis une analyse à la fois précise, robuste et interprétable.

Les résultats ne sont malheureusement pas très fiable car il y a beaucoup de valeurs manquantes dans le jeu de données. Nous avons préféré supprimé les lignes impliquées plutôt que de combler avec des données artificielles.
